# 📊 Superstore Data Analysis using Pandas

## 📌 Project Overview
This project performs exploratory data analysis (EDA) on the Superstore dataset using Python Pandas. The goal is to understand sales performance, profit distribution, customer behavior, and business trends through data cleaning, transformation, aggregation, and visualization.

---

## 📂 Dataset
The dataset used is the **Superstore Dataset** from Kaggle:  
https://www.kaggle.com/datasets/vivek468/superstore-dataset-final

It contains information about:
- Orders
- Customers
- Products
- Sales
- Profit
- Regions and Segments

---

## 🎯 Objectives
- Load and explore real-world business data
- Clean and preprocess dataset
- Perform feature engineering
- Analyze sales and profit trends
- Use grouping and pivot tables for insights
- Visualize key patterns using charts

---

## 🛠️ Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib

---

## 🔍 Key Steps

### 1. Data Loading & Exploration
- Loaded dataset using Pandas
- Checked structure, data types, and summary statistics

### 2. Data Cleaning
- Identified missing values
- Handled missing data
- Converted date columns to datetime format

### 3. Feature Engineering
- Extracted Year and Month from order date
- Created new feature (Profit Ratio)

### 4. Data Analysis
- Sales by Category
- Profit by Region
- Top 5 Customers
- Monthly sales trends

### 5. Pivot Tables
- Sales by Category and Region
- Sales by Segment over time
- Profit by Sub-Category

### 6. Data Splitting & Merging
- Split dataset into smaller DataFrames
- Merged them back using Pandas merge

### 7. Data Visualization
- Line chart: Sales trend over time
- Bar chart: Sales by category
- Bar chart: Profit by region
- Pie chart: Segment distribution

---

## 📊 Key Insights
- Technology is the most profitable category
- Some regions generate low profit despite high sales
- Sales show seasonal/monthly patterns
- A small group of customers contributes most of the revenue

---

## 🚀 Conclusion
This project demonstrates how Pandas can be used for real-world business data analysis. It helps in understanding how data-driven decisions can improve business performance through insights into sales, profit, and customer behavior.
